Important Faq

What is the Document Object Model (DOM)?

DOM means Document Object Model. it is the HTML document and tree structured. Every Element is node in this tree structre.

How do you select an element from the DOM?

There are couple of ways to select and element from the DOM. 1.getElementById 2.getElementsByClassName 3.querySelector 4.querySelectorAll 5.getElementsByTagName

What is event delegation in the context of the DOM, and why is it useful?

When an event occurs in child element it reaches to the top of the document.if we add event listener every single element under a parent than it will hamper the efficiecy. then we attached event listener in parent.so, that any event occurs in child it can be tracted from parents.

How do you manipulate an element's attributes and styles using the DOM?

First of all we have to select and element than use getAttribute method to get the attributes. it will return us the value of the attributes. We can the change the value. Example: src attritube. const image = document.getElementById("image"); const src = image.getAttribute("src"); image.setAttribute("src", "./assets/coin.png");
